PASO 3: CONFIGURAR EL TOKEN
============================

En tu login o donde guardas el usuario, asegúrate que guardes el JWT:

localStorage.setItem('authToken', tuTokenJWT);

El código busca el token así:
const token = localStorage.getItem('authToken');

CUANDO ALGO NO FUNCIONE
=======================

Abre la consola del navegador (F12 -> Console) y busca estos errores:

1. "No token found"
   → Falta guardar el token en localStorage
   → Solución: localStorage.setItem('authToken', TOKEN)

2. "POST /api/inventario/bulk/productos 401"
   → Token inválido o expirado
   → Solución: Vuelve a iniciar sesión

3. "POST /api/inventario/bulk/productos 400"
   → Faltan productos o están vacíos
   → Verifica que escribiste el nombre del producto

4. "GET /api/inventario/productos/.../setup 404"
   → El producto se creó pero no se encuentra
   → Verifica que el backend está corriendo en http://localhost:3000

5. Nada aparece en la consola pero no funciona
   → El archivo JS no se cargó correctamente
   → Verifica que <script src="CODIGO_FRONTEND_LISTO.js"></script> está en el HTML
